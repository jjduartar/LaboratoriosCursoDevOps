## Instructions

1. Install [docker-compose](https://docs.docker.com/compose/install/)

2. Create a folder to run this demo

```
mkdir democompose
```

3. Let's create the docker-compose.yml file
   
4. Run docker-compose

```
docker-compose up
```

4. Test the different applications
